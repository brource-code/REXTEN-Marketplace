# 🌐 URL для доступа к приложению

## ✅ Все варианты работают

### Через localhost (без домена)

1. **HTTP с редиректом:**
   ```
   http://localhost
   ```
   Автоматически перенаправит на `https://localhost:8443`

2. **HTTPS напрямую:**
   ```
   https://localhost:8443
   ```

### Через домен rexten.local

1. **HTTP с редиректом:**
   ```
   http://rexten.local
   ```
   Автоматически перенаправит на `https://rexten.local:8443`

2. **HTTPS напрямую:**
   ```
   https://rexten.local:8443
   ```

## ⚠️ Предупреждение о сертификате

Браузер покажет предупреждение о самоподписанном сертификате. Это нормально:

- **Chrome/Edge**: "Дополнительно" → "Перейти на сайт (небезопасно)"
- **Firefox**: "Дополнительно" → "Принять риск и продолжить"
- **Safari**: "Показать подробности" → "Посетить этот веб-сайт"

## 🔍 Проверка работы

Все URL проверены и работают:
- ✅ `http://localhost` → редирект на HTTPS
- ✅ `https://localhost:8443` → HTTP 302
- ✅ `http://rexten.local` → редирект на HTTPS
- ✅ `https://rexten.local:8443` → HTTP 302

## 📝 Прямой доступ к серверам (без nginx)

Если nginx не нужен:
- **Frontend**: http://localhost:3003
- **Backend API**: http://localhost:8000/api
