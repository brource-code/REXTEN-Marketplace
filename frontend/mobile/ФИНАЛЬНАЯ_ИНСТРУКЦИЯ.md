# 🎯 Финальная инструкция

## ✅ Expo перезапускается на порту 8081

Подождите несколько секунд, пока Expo запустится.

## 📱 Затем в Expo Go на iPhone:

### Способ 1: Ввести URL вручную (рекомендуется)

1. Откройте **Expo Go**
2. Нажмите **"Enter URL manually"**
3. Введите: **`exp://192.168.1.72:8081`**
4. Нажмите **"Connect"**

### Способ 2: Через веб-интерфейс

1. Откройте в браузере: **http://localhost:8081**
2. Там будет QR-код
3. Отсканируйте его в Expo Go

### Способ 3: Через приложение Камера iOS

1. Откройте стандартное приложение **Камера** на iPhone
2. Наведите на QR-код в терминале
3. Нажмите на уведомление
4. Выберите "Open in Expo Go"

---

## ⏱ Подождите 10-15 секунд

Expo сейчас запускается. Через несколько секунд в терминале появится QR-код и меню.

---

## ✅ Готово!

После того как увидите QR-код в терминале - используйте любой из способов выше.
