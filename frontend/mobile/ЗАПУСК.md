# 🚀 Как запустить и увидеть QR-код

## ✅ Простой способ (рекомендуется):

В терминале выполните:

```bash
cd "/Users/turbin/Desktop/REXTEN marketplace/ecme-admin/mobile"
npx expo start
```

**После запуска:**
1. В терминале появится меню с опциями
2. Нажмите **`s`** для включения Tunnel mode (QR-код появится)
3. ИЛИ нажмите **`i`** для iOS симулятора (если Mac)
4. ИЛИ откройте в браузере: **http://localhost:8081** (там будет QR-код)

---

## 📱 Подключение с iPhone:

1. **Установите Expo Go** из App Store
2. **Откройте Expo Go** → "Scan QR code"
3. **Отсканируйте QR-код** из терминала или браузера

---

## 🌐 Веб-интерфейс (самый надежный способ):

Просто откройте в браузере:
```
http://localhost:8081
```

Там будет:
- ✅ QR-код для сканирования
- ✅ URL для ручного ввода
- ✅ Все инструкции

---

## 💡 Горячие клавиши в терминале Expo:

- **`s`** - включить Tunnel mode (QR-код появится)
- **`i`** - открыть в iOS симуляторе (Mac)
- **`a`** - открыть в Android эмуляторе
- **`w`** - открыть в веб-браузере
- **`r`** - перезагрузить приложение
