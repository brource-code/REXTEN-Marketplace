# ✅ Простое решение (без Tunnel)

## Запустите Expo в обычном режиме:

```bash
cd "/Users/turbin/Desktop/REXTEN marketplace/ecme-admin/mobile"
npx expo start
```

## Затем в Expo Go на iPhone:

1. Откройте **Expo Go**
2. Нажмите **"Enter URL manually"**
3. Введите: **`exp://192.168.1.72:8081`**
4. Нажмите **"Connect"**

---

## Или откройте в браузере:

```
http://localhost:8081
```

Там будет QR-код, который точно работает!

---

## Почему это работает:

- Вы в одной сети ✅
- Tunnel не нужен ✅
- Просто введите URL вручную ✅
