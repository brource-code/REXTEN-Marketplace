# 🔧 Решение проблемы "запустить npx expo start"

## ✅ Решение 1: Ввести URL вручную (самый надежный)

1. Откройте **Expo Go** на iPhone
2. Нажмите **"Enter URL manually"** (Ввести URL вручную)
3. Введите:
   ```
   exp://192.168.1.72:8081
   ```
4. Нажмите **"Connect"**

---

## ✅ Решение 2: Использовать Tunnel mode

В терминале Expo нажмите клавишу **`s`**

Это создаст новый QR-код с Tunnel mode, который работает надежнее.

---

## ✅ Решение 3: Проверить подключение

Убедитесь что:
- ✅ iPhone и компьютер в **одной Wi-Fi сети**
- ✅ Expo запущен (видно QR-код в терминале)
- ✅ Порт 8081 не заблокирован файрволом

---

## ✅ Решение 4: Перезапустить Expo

1. В терминале нажмите **Ctrl+C** (остановить Expo)
2. Запустите заново:
   ```bash
   cd "/Users/turbin/Desktop/REXTEN marketplace/ecme-admin/mobile"
   npx expo start --tunnel
   ```
3. Отсканируйте новый QR-код

---

## 🎯 Рекомендуемый порядок действий:

1. **Попробуйте Решение 1** (ввести URL вручную) - это самый быстрый способ
2. Если не работает - **Решение 2** (Tunnel mode, нажмите `s`)
3. Если все еще не работает - **Решение 4** (перезапуск с tunnel)
