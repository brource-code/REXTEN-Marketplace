# Анализ проблемы с компаниями и объявлениями

## Текущая ситуация

### Проблемы:
1. **Нет связи между объявлением и компанией на фронтенде**
   - На странице объявления (`/marketplace/[slug]`) нет информации о том, что это объявление компании
   - Нет кнопки "Все объявления компании" или "Открыть компанию"
   - Пользователь не понимает, что у компании может быть несколько объявлений

2. **Путаница с идентификацией**
   - Объявления имеют свой `link` (slug), который может не совпадать со slug компании
   - Компания имеет свой `slug`, но он не используется для перехода к списку объявлений
   - Нет единой точки входа для просмотра всех объявлений компании

3. **Отсутствие навигации**
   - Нет страницы компании, которая показывает все её объявления
   - Нет возможности перейти от объявления к компании и обратно
   - Нет фильтрации объявлений по компании на странице `/services`

## Рекомендации по исправлению

### 1. Добавить информацию о компании на странице объявления

**На странице `/marketplace/[slug]`:**

```jsx
// После названия объявления добавить:
<div className="flex items-center gap-2 mb-3">
    <span className="text-sm text-gray-500 dark:text-gray-400">
        Компания:
    </span>
    <Link 
        href={`/marketplace/company/${business.companySlug || business.companyId}`}
        className="text-sm font-semibold text-blue-600 dark:text-blue-400 hover:underline"
    >
        {business.companyName}
    </Link>
    {companyAdvertisementsCount > 1 && (
        <span className="text-xs text-gray-400">
            ({companyAdvertisementsCount} объявлений)
        </span>
    )}
</div>
```

### 2. Создать страницу компании `/marketplace/company/[slug]`

**Новый роут:** `/marketplace/company/[slug]`

**Функционал:**
- Показывает информацию о компании
- Список всех активных объявлений компании
- Рейтинг компании (из всех отзывов)
- Общая статистика

**Backend API endpoint:**
```php
// GET /api/marketplace/company/{slug}
public function getCompanyProfile($slug)
{
    $company = Company::where('slug', $slug)
        ->orWhere('id', $slug)
        ->where('status', 'active')
        ->where('is_visible_on_marketplace', true)
        ->with('advertisements')
        ->first();
    
    // Возвращает:
    // - Информацию о компании
    // - Список всех объявлений
    // - Общий рейтинг
    // - Статистику
}
```

// Для объявлений добавить:
