# 🚀 Быстрая инструкция для iPhone

## ✅ Шаг 1: Установите Expo Go
Откройте **App Store** на iPhone и установите **"Expo Go"** (бесплатно)

## ✅ Шаг 2: Найдите QR-код
1. Откройте **терминал**, где запущен Expo (где вы видели `npm run start`)
2. **Прокрутите вверх** - там должен быть QR-код
3. Если не видите - в терминале Expo нажмите клавишу **`s`** (для Tunnel mode)

## ✅ Шаг 3: Отсканируйте
1. Откройте **Expo Go** на iPhone
2. Нажмите **"Scan QR code"**
3. Наведите камеру на QR-код из терминала
4. Готово! 🎉

---

## 🔄 Если QR-код не виден:

### Вариант 1: Tunnel mode
В терминале Expo нажмите **`s`** - появится новый QR-код

### Вариант 2: Вручную
1. В терминале найдите строку типа: `exp://192.168.1.72:8081`
2. В Expo Go нажмите "Enter URL manually"
3. Введите этот URL

### Вариант 3: iOS симулятор (если Mac)
В терминале Expo нажмите **`i`** - откроется в симуляторе

---

- IP вашего компьютера: **192.168.1.72**
- Файл `.env` обновлен автоматически
