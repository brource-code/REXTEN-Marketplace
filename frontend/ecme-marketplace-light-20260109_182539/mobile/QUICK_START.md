# Быстрый старт

## 🚀 Приложение запущено!

Expo dev server должен быть запущен. Вы увидите QR-код в терминале.

## Как подключиться:

### Для iPhone (рекомендуется):

1. **Установите Expo Go** из App Store (если еще не установлено)
2. **Откройте Expo Go** на iPhone
3. **Отсканируйте QR-код** из терминала
   - Или нажмите `i` в терминале для открытия в iOS симуляторе

### Альтернативные способы:

- **iOS симулятор**: Нажмите `i` в терминале Expo
- **Android эмулятор**: Нажмите `a` в терминале Expo
- **Веб-версия**: Нажмите `w` в терминале Expo

## ⚙️ Настройка API

Если API не работает, проверьте файл `.env`:

```bash
# Для iOS симулятора
EXPO_PUBLIC_API_BASE_URL=http://localhost:8000/api

# Для физического устройства (замените на IP вашего компьютера)
EXPO_PUBLIC_API_BASE_URL=http://192.168.1.XXX:8000/api
```

## 🔍 Что проверить:

1. ✅ Приложение открывается
2. ✅ Главный экран каталога загружается
3. ✅ Поиск работает
4. ✅ Фильтры открываются
5. ✅ Карточки услуг отображаются
6. ✅ Переход на детальную страницу работает

## 🐛 Если что-то не работает:

1. **Ошибки API**: Проверьте, что Laravel API запущен и доступен
2. **CORS ошибки**: Настройте CORS на Laravel бэкенде
3. **Изображения не загружаются**: Проверьте URL изображений
4. **Навигация не работает**: Проверьте, что все зависимости установлены

## 📱 Горячие клавиши в Expo:

- `r` - перезагрузить приложение
- `m` - открыть меню разработчика
- `i` - открыть в iOS симуляторе
- `a` - открыть в Android эмуляторе
- `w` - открыть в веб-браузере
