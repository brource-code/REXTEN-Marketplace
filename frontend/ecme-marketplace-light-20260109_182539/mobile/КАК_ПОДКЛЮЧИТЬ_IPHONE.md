# 📱 Как запустить на iPhone

## Способ 1: Через QR-код (самый простой)

### Шаг 1: Найдите QR-код
QR-код отображается в **терминале**, где запущен Expo. Выглядит примерно так:

```
┌─────────────────────────────────┐
│  ████████████████████████████  │
│  ████████████████████████████  │
│  ████████████████████████████  │
│  ████████████████████████████  │
│  ████████████████████████████  │
│  ████████████████████████████  │
│  ████████████████████████████  │
└─────────────────────────────────┘
```

**Где искать:**
- Откройте терминал, где вы запустили `npm run start` или `npx expo start`
- Прокрутите вверх, если нужно
- QR-код должен быть виден сразу после запуска

### Шаг 2: Установите Expo Go
1. Откройте **App Store** на iPhone
2. Найдите и установите **"Expo Go"** (бесплатное приложение)

### Шаг 3: Отсканируйте QR-код
1. Откройте приложение **Expo Go** на iPhone
2. Нажмите **"Scan QR code"** (Сканировать QR-код)
3. Наведите камеру на QR-код в терминале
4. Приложение автоматически загрузится!

---

## Способ 2: Через URL (если QR-код не виден)

### Шаг 1: Найдите URL в терминале
В терминале Expo вы увидите что-то вроде:
```
Metro waiting on exp://192.168.1.XXX:8081
```

### Шаг 2: Откройте в Expo Go
1. Откройте **Expo Go** на iPhone
2. Нажмите **"Enter URL manually"** (Ввести URL вручную)
3. Введите URL из терминала (например: `exp://192.168.1.XXX:8081`)

---

## Способ 3: Через Tunnel (если не работает локальная сеть)

### В терминале Expo нажмите:
```
s
```
Это переключит на **Tunnel mode** - будет работать даже если iPhone и компьютер в разных сетях.

После этого появится новый QR-код - отсканируйте его.

---

## Способ 4: Через iOS симулятор (на Mac)

Если у вас Mac, можете запустить прямо в симуляторе:

### В терминале Expo нажмите:
```
i
```

Это откроет приложение в iOS симуляторе на вашем Mac.

---

## ❗ Важно для физического iPhone:

Если используете **реальный iPhone** (не симулятор), убедитесь что:

1. **iPhone и компьютер в одной Wi-Fi сети**
2. **В файле `.env` указан IP вашего компьютера**, а не localhost:

```bash
# Узнайте IP вашего Mac:
# В терминале выполните: ifconfig | grep "inet " | grep -v 127.0.0.1

# Затем в .env укажите:
EXPO_PUBLIC_API_BASE_URL=http://192.168.1.XXX:8000/api
```

---

## 🔍 Если QR-код не появляется:

1. **Остановите Expo** (Ctrl+C в терминале)
2. **Запустите заново** с очисткой кэша:
   ```bash
   cd mobile
   npx expo start --clear
   ```

3. **Или используйте Tunnel mode** (нажмите `s` в терминале Expo)

---

## 📞 Быстрая помощь:

- **QR-код не виден?** → Нажмите `s` для Tunnel mode
- **Не подключается?** → Проверьте, что iPhone и Mac в одной Wi-Fi сети
- **Ошибки API?** → Проверьте файл `.env` и IP адрес
