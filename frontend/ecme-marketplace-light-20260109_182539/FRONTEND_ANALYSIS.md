# Аналитика фронтенда Ecme Marketplace

## 📊 Обзор всех представлений

### ✅ Публичная часть `/(public)`

#### Страницы:
1. **`/` (root)** → редирект на `/services` ✅
2. **`/landing`** - Главная страница ✅
3. **`/services`** - Каталог услуг с фильтрами ✅
4. **`/marketplace/[slug]`** - Профиль услуги/бизнеса ✅
5. **`/business/[slug]`** - Профиль бизнеса (если отличается от marketplace) ⚠️
6. **`/profile`** - Профиль клиента (отзывы, избранное, скидки, уведомления) ✅
7. **`/profile/settings`** - Настройки профиля ✅
8. **`/access-denied`** - Страница отказа в доступе ✅
9. **`/sign-in`** - Вход для клиентов ✅
10. **`/sign-up`** - Регистрация ✅
11. **`/forgot-password`** - Восстановление пароля ✅
12. **`/reset-password`** - Сброс пароля ✅

#### Навигация:
- **PublicNavbar**: Логотип → `/landing`, Вход → `/sign-in`, Регистрация → `/sign-up`
- **Для авторизованных клиентов**: Профиль → `/profile`, Уведомления (в навбаре)
- **Ссылки на профиль**: Из навбара → `/profile`

### ✅ Личный кабинет клиента

#### Страницы:
1. **`/profile`** (публичный) - Основной профиль с вкладками ✅
   - Вкладки: Отзывы, Избранное, Скидки и бонусы, Уведомления
   - Ссылка на `/profile/settings`
   
2. **`/profile/settings`** - Настройки профиля ✅
   - Форма редактирования
   - Загрузка аватара
   - Ссылка назад на `/profile`

3. **`/client/orders`** - История заказов ✅
   - Список заказов
   - Фильтры (статус, дата)
   - Детали заказа (модалка)
   - ⚠️ **ПРОБЕЛ**: Нет ссылки из профиля/навбара

4. **`/client/booking`** - Бронирования ✅
   - Список активных/прошедших бронирований
   - Календарь
   - Отмена/повторное бронирование
   - ⚠️ **ПРОБЕЛ**: Нет ссылки из профиля/навбара

5. **`/client/profile`** - Старая страница профиля ⚠️
   - ⚠️ **ПРОБЕЛ**: Дублирует `/profile`, не используется

#### Навигация:
- ⚠️ **ПРОБЕЛ**: Нет прямых ссылок на `/client/orders` и `/client/booking` из навбара или профиля
- Ссылки из бронирований: "Перейти к бизнесу" → `/marketplace/[slug]`
- Ссылки из заказов: "Перейти к бизнесу" → `/marketplace/[slug]`

## 🔗 Анализ связей между страницами

### Публичная часть → Личный кабинет клиента

✅ **Работает:**
- `/services` → `/marketplace/[slug]` (клик на услугу)
- `/marketplace/[slug]` → Бронирование (форма на странице)
- Навбар → `/profile` (для авторизованных клиентов)
- `/profile` → `/profile/settings`

⚠️ **Пробелы:**
- Нет ссылок на `/client/orders` из профиля или навбара
- Нет ссылок на `/client/booking` из профиля или навбара
- Нет ссылок из заказов/бронирований обратно в профиль

### Личный кабинет клиента → Публичная часть

✅ **Работает:**
- `/client/booking` → `/marketplace/[slug]` (кнопка "Перейти к бизнесу")
- `/client/orders` → `/marketplace/[slug]` (кнопка "Перейти к бизнесу")
- `/profile` → `/marketplace/[slug]` (из избранного)

⚠️ **Пробелы:**
- Нет ссылки "Назад" из `/client/orders` в профиль
- Нет ссылки "Назад" из `/client/booking` в профиль

### Админка бизнеса → Публичная часть

✅ **Работает:**
- `/business/clients` → `/marketplace/[slug]` (если есть ссылка на профиль бизнеса)

⚠️ **Пробелы:**
- Нет прямых ссылок на публичный профиль бизнеса из админки

### Суперадминка → Другие части

✅ **Работает:**
- `/superadmin/companies` → `/marketplace/[slug]` (если есть)
- `/superadmin/users` → `/profile` (если клиент)

## 🚨 Найденные пробелы

### 1. Навигация клиента

#### Проблема:
- Нет доступа к `/client/orders` и `/client/booking` из навбара или профиля
- Страницы `/client/orders` и `/client/booking` изолированы

#### Решение:
- Добавить ссылки в навбар (dropdown меню клиента)
- Или добавить вкладки в профиль
- Или добавить быстрые ссылки в профиль

### 2. Дублирование профиля

#### Проблема:
- `/client/profile` - старая страница (простая форма)
- `/profile` - новая страница (с вкладками)
- Обе страницы существуют

#### Решение:
- Удалить `/client/profile` или сделать редирект на `/profile`
- Обновить все ссылки

### 3. Обратная навигация

#### Проблема:
- Нет кнопки "Назад" из `/client/orders` и `/client/booking`
- Нет хлебных крошек

#### Решение:
- Добавить хлебные крошки
- Или добавить кнопку "Назад в профиль"

### 4. Связи между бизнес-панелью и публичным профилем

#### Проблема:
- Нет прямых ссылок на публичный профиль бизнеса из админки

#### Решение:
- Добавить кнопку "Просмотр профиля" в настройках бизнеса
- Добавить ссылку в дашборде

### 5. Уведомления в дашборде бизнеса

#### Проблема:
- Уведомления помечены как заглушка

#### Решение:
- Добавить блок уведомлений в дашборд
