## 🔧 Интеграция

### Шаг 1: Обновить .env.local

Добавьте или обновите в `.env.local`:

```env
NEXT_PUBLIC_LARAVEL_API_URL=http://localhost:8000/api
NEXT_PUBLIC_USE_MOCK_AUTH=false
```

### Шаг 2: Запустить Backend

```bash
cd backend
php artisan serve
```

Backend будет доступен на `http://localhost:8000`

### Шаг 3: Перезапустить Frontend

```bash
npm run dev
```
