# Автоматические тесты

## Тесты для объявлений

Автоматические тесты для проверки функциональности создания, редактирования и управления объявлениями.

### Запуск тестов

1. **Убедитесь, что Laravel сервер запущен:**
   ```bash
   cd backend
   php artisan serve
   ```

2. **Запустите тесты:**
   ```bash
   npm run test:advertisements
   ```

### Переменные окружения

Вы можете настроить тесты через переменные окружения:

```bash
API_URL=http://localhost:8000/api \
TEST_EMAIL=admin@ecme.com \
TEST_PASSWORD=demo12345 \
npm run test:advertisements
```

### Что тестируется

1. ✅ Авторизация суперадмина
2. ✅ Получение списка компаний
3. ✅ Загрузка изображения
4. ✅ Создание рекламного объявления (с датами)
5. ✅ Создание обычного объявления (без дат)
6. ✅ Получение объявления по ID
7. ✅ Обновление объявления
8. ✅ Получение списка объявлений
9. ✅ Одобрение объявления
10. ✅ Удаление объявления

### Результаты

Тесты выводят цветные результаты:
- 🟢 Зеленый - тест пройден
- 🔴 Красный - тест провален
- 🟡 Желтый - информационное сообщение
- 🔵 Синий - заголовок теста

В конце выводится итоговая статистика:
- Всего тестов
- Пройдено
- Провалено
